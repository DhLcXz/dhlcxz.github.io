# 个人网站

这是一个现代化、响应式的个人网站，使用纯前端技术（HTML、CSS、JavaScript）构建，无需任何框架即可运行。

## 功能特点

- ✅ **响应式设计**：完美适配电脑、平板和手机
- ✅ **现代简约风格**：采用最新的设计趋势和色彩方案
- ✅ **完整功能模块**：
  - 导航栏（支持移动端菜单）
  - 个人简介区域
  - 技能展示（带动画效果）
  - 作品案例展示
  - 联系方式表单
  - 页脚和订阅功能
- ✅ **交互功能**：
  - 平滑滚动导航
  - 返回顶部按钮
  - 表单验证
  - 技能条动画
  - 移动端菜单切换

## 文件结构

```
个人网站/

├── css/
```

## 如何自定义

### 1. 修改个人信息
打开 `index.html` 文件，修改以下内容：
- 姓名和标题（第15行）
- 个人简介（第47-54行）
- 联系信息（第166-178行）
- 统计数据（第57-72行）

### 2. 修改技能
在 `index.html` 中找到技能部分（第75-146行），可以：
- 添加/删除技能类别
- 修改技能名称和百分比
- 调整技能条颜色（在CSS中修改）

### 3. 修改作品案例
在 `index.html` 中找到作品部分（第149-200行），可以：
- 添加/删除作品项目
- 修改项目描述和技术栈标签
- 替换图标（使用Font Awesome图标）

### 4. 修改样式
打开 `css/style.css` 文件，可以：
- 修改颜色方案（第2-11行的CSS变量）
- 调整字体和间距
- 修改响应式断点

### 5. 添加图片
将个人照片放入 `images/` 文件夹，然后在 `index.html` 中替换第40行的图标占位符：
```html
<!-- 替换这个 -->
<div class="image-placeholder">
    <i class="fas fa-user-circle"></i>
</div>

<!-- 为这个 -->
<div class="hero-image">
    <img src="images/your-photo.jpg" alt="你的姓名" class="profile-photo">
</div>
```

## 如何运行

1. 直接双击 `index.html` 文件在浏览器中打开
2. 或者使用本地服务器（推荐）：
   ```bash
   # 使用Python
   python -m http.server 8000
   
   # 或使用Node.js
   npx serve .
   ```
   然后在浏览器中访问 `http://localhost:8000`
